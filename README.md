## レポート

# 処理説明

Opencvで画像取得後フーリエ変換の関数を用いてフーリエ変換を行う。
処理後の画像をfigure1.jpgに張り付ける。
その後figure1.jpgを逆フーリエ変換したものをfigure2.jpgに張り付る。
マウスでの操作はわからなかったので実装できなかった。
